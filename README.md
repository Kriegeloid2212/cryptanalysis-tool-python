# Криптоаналіз історичних шифрів (Historical Ciphers Cryptanalysis)

Програмний застосунок із графічним інтерфейсом користувача (GUI) для шифрування, дешифрування та автоматизованого криптоаналізу класичних історичних шифрів. 

Проект виконано в межах курсової роботи з дисциплін циклу професійної підготовки.

## 🔑 Ключовий функціонал

Застосунок підтримує роботу з двома мовними алфавітами (українським та англійським) і реалізує наступні криптографічні системи та інструменти аналізу:

1.  **Шифр Цезаря:**
    *   Шифрування та дешифрування з довільним числовим кроком (зсувом).
    *   Автоматизований криптоаналіз методом «грубої сили» (Brute Force) з інтерактивним вибором результату з таблиці.
2.  **Шифр Атбаш:**
    *   Моноалфавітна заміна за принципом «дзеркального» алфавіту (реалізовано як самообернена функція).
3.  **Шифр Віженера:**
    *   Поліалфавітна система шифрування за ключовим словом.
    *   Оцінка довжини ключа на основі розрахунку індексу збігів (Index of Coincidence) з побудовою графіку на Canvas.
4.  **Шифр Плейфера:**
    *   Біграмне шифрування за ключовим словом.
    *   Динамічна генерація та візуалізація ключової матриці розміром 5х5 (для англійської мови) або 6х6 з додаванням цифр (для української мови).
5.  **Модуль частотного аналізу:**
    *   Побудова гістограми частотності літер досліджуваного тексту та порівняння її із еталонною мовною нормою (для української та англійської мов).
